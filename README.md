A simple **HTML + CSS project** that creates a clean, animated **loading spinner**. The design uses border styling and keyframe animations to rotate a circle, making it ideal for loading indicators in web applications.

## ‚ú® Features
- **Circular Spinner Design**  
  - A 100√ó100px circle with a thick border.  
  - Neutral base color (`grey`) with an active highlight (`purple`) on the top border.  

- **Smooth Animation**  
  - Spins continuously using `@keyframes spin`.  
  - Rotation completes every 1.5 seconds for a fluid effect.  

- **Customizable Variables**  
  - Colors defined in `:root` for easy theme adjustments.  
  - Spinner size and border thickness can be modified for different use cases.  

- **Responsive Centering**  
  - Flexbox centers the loader both vertically and horizontally on the page.  

- **Minimal Structure**  
  - Simple HTML with `.container` and `.loader` elements.  
  - Lightweight CSS for styling and animation.
